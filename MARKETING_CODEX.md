## 2. VISUAL SYSTEM (The "Retina")

### A. Color Palette
*   **The Void (Backgrounds):**
    *   `bg-zinc-950` (#09090b) - Main background.
    *   `bg-black` (#000000) - Deep contrast.
*   **The Signal (Primary):**
    *   `text-emerald-500` (#10b981) - Success, Code, "Alive".
    *   `text-emerald-400` (#34d399) - Glow effects.
*   **The Anomaly (Accents):**
    *   `text-fuchsia-500` (#d946ef) - Recursion, Special Events.
    *   `text-rose-500` (#f43f5e) - Errors, "Harm Inhibition".

### B. Typography
*   **Headings:** `font-sans` (Inter). Weights: 700 (Bold), 900 (Black). Tracking: `tracking-tight`.
*   **Body:** `font-sans` (Inter). Weights: 400 (Regular). Text color: `text-zinc-400`.
*   **Data/Code:** `font-mono`. Used for: Console logs, "Thinking" steps, technical specs.

### C. Effects
*   **The Glitch:** CSS animations that briefly distort the UI on hover or error.
*   **The Pulse:** Subtle breathing glow on active elements (`animate-pulse`).
*   **The Terminal:** Typing effects (`TypewriterCode`) for all AI output.
