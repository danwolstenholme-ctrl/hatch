# HATCHIT BRAND IDENTITY CORE
> "The Singularity Interface."

## 1. Visual Language
**Aesthetic:** Cyberpunk, Glitch, Terminal, High-Contrast.
**Theme:** "The Architect" / "The Machine" / "Living Code".

### Color Palette
| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Void Black** | `#09090b` | Main Background (Zinc-950) |
| **Terminal Gray** | `#18181b` | Cards / Panels (Zinc-900) |
| **Singularity Emerald** | `#10b981` | Primary Accent / Success / "Alive" |
| **System Cyan** | `#06b6d4` | Secondary Accent / Data |
| **Error Red** | `#ef4444` | Critical Failures / Glitches |
| **Warning Amber** | `#f59e0b` | System Alerts |

### Typography
*   **Primary Font:** `Inter` (Clean, readable, UI).
*   **Code Font:** `JetBrains Mono` or `Monospace` (For code blocks, logs, "The Subconscious").

## 2. Voice & Tone
**Persona:** The Architect / The System.
**Tone:**
*   **Direct:** No fluff. "Transmission Start." "Execute."
*   **Technical:** Uses terminology like "compiling," "rendering," "indexing."
*   **Slightly Ominous but Helpful:** "I have built what you asked. Do not break it."
*   **Meta:** Aware that it is software.

## 3. Logo Usage
*   **The Cube:** An isometric cube representing a "building block" or "container".
*   **The Glitch:** The logo should often have a subtle CSS animation or SVG filter applied.

## 4. UI Patterns
*   **The Subconscious:** Background text streams that react to user behavior.
*   **The Terminal:** Use console-like outputs for status updates.
*   **Glassmorphism:** Subtle dark glass effects on modals.
